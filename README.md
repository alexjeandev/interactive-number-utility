## Interactive Number Utility

A simple web-based utility built with vanilla JavaScript that allows users to
increment, decrement, and reset a number.

## Concepts Used
- DOM manipulation
- Event listeners
- User input handling
- Application state

## Technologies
- HTML
- CSS
- JavaScript

## Development Workflow
- Built and tested locally using the macOS Terminal
- Version control with Git
- Authenticated and pushed to GitHub using SSH
- Deployed as a live site using GitHub Pages
